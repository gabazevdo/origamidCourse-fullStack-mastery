  1. EXERCÍCIO aula Variáveis
    1. Declarar uma variável com o seu nome
    2. Declarar uma variável com a sua idade
    3. Declarar uma variável com a sua comida
    4. Variável da comida favorita, mostre e não atribua valor
    5. Declarar 5 variáveis diferentes e não atribuir valor

  2. EXERCÍCIO aula Tipos de dados
    1. Declare uma variável contendo uma string
    2. Declare uma variável contendo um número dentro de uma string
    3. Declare uma variável com a sua idade
    4. Declare duas variáveis, uma com seu nome...
    5. ... e outra com seu sobrenome e some as mesmas
    6. Coloque a seguinte frase em uma variável: It's time
    7. Verifique o tipo da variável que contém o seu nome

  3. Exercício aula Números e operadores
    1. Qual o resultado da seguinte expressão?
    2. Crie duas expressões que retornem NaN
    3. Somar a string '200' com o número 50 e retornar 250
    4. Incremente o número 5 e retorne o seu valor incrementado
    5. Como dividir o peso por 2?

  4. Exercícios aula Boolean e Condicionais
    1. Verifique se a sua idade é maior do que a de algum parente
    2. Dependendo do resultado coloque no console 'É maior', 'É igual' ou 'É menor'
    3. Qual valor é retornado na seguinte expressão?
    4. Verifique se as seguintes variáveis são Truthy ou Falsy.
    5. Compare o total de habitantes do Brasil com China (valor em milhões)
    6. O que irá aparecer no console?

  5. Exercícios aula Funções
    1. Crie uma função para verificar se um valor é Truthy
    2. Crie uma função matemática que retorne o perímetro de um quadrado
    3. Crie uma função que retorne o seu nome completo
    4. Crie uma função que verifica se um número é par
    5. Crie uma função que retorne o tipo de dado do argumento passado nela (typeof)
    6. Utilize a função addEventListener
      addEventListener é uma função nativa do JavaScript
      o primeiro parâmetro é o evento que ocorre e o segundo o Callback
      utilize essa função para mostrar no console o seu nome completo
      quando o evento 'scroll' ocorrer.
    7. Corrija os erros da função

  6. Exercícios aula Objetos
    1. Crie um objeto com os seus dados pessoais
    2. Deve possui pelo menos duas propriedades nome e sobrenome
    3. Modifique o valor da propriedade preco para 3000
    4. Crie um objeto de um cachorro que represente um labrador, preto com 10 anos, que late ao ver um homem
